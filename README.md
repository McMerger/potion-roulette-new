# Potions Roulette - Web Version (Dark Alchemy)

This is a web-based implementation of Potions Roulette, designed for maximum emotional impact and a "Dark Alchemy" aesthetic.

## Tech Stack
- **Three.js**: Moody 3D background with floating table and particle effects.
- **Vanilla JavaScript**: Game logic and state management.
- **CSS3**: Animations, glassmorphism, and responsive UI.

## How to Run
1.  Navigate to this directory.
2.  Open `index.html` in any modern web browser.
    *   *Note: Due to ES Module security (CORS), it is recommended to open this via a local server (e.g., Live Server extension in VS Code, or `npx serve .`).*

## Features Ported
- **Full Game Loop:** Crafting, Choosing, and Resolving potions.
- **Visual Feedback:** 2D Screen shakes, color-coded flashes, and dynamic HUD.
- **Chaos Wheel:** A tension-filled physical spin animation for double-chaos outcomes.
- **Zero Local Install:** All dependencies are loaded via CDN (imports using Three.js module).
